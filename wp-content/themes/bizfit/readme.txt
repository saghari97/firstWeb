== Description ==
Bizfit, a business WordPress theme that suits for small to large agencies. Not a single coding required for building a website. Pre-define templates help to build your website quickly. This theme provides extra options on Header, Footer, Post, Sidebar, etc. that provide endless possibilities to make your website awesome.

For more information about BizFit please go to https://suitabletheme.com/

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
